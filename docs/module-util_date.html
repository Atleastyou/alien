<static> format(format, date, config)

格式化日期
主要参考ECMA规范定义：YYYY-MM-DDTHH:mm:ss.sssZ
其他参数参考自moment
Parameters/Arguments:
Name Type Argument Description
format   String 格式化字符串
假设当前时间为：2014年1月1日 19点9分9秒9毫秒 周三
日期
至少4位年份YYYY 2014
至少2位年份YY 14
至少2位月份MM 01
至少1位月份M 1
至少2位日期DD 01
至少1位日期D 1
时间
至少2位24小时制小时HH 19
至少1位24小时制小时H 19
至少2位12小时制小时hh 07
至少1位12小时制小时h 7
至少2位分钟数mm 09
至少1位分钟数m 9
至少2位秒数ss 09
至少1位秒数s 9
至少3位毫秒数SSS 009
至少2位毫秒数SS 09
至少1位毫秒数S 9
时段
星期e 三
上下午a 下午
date   Date |   Object |   Number |   String <optional>
 日期
config   Object <optional>
 格式配置
Returns:
  null |   string
Example
// 默认的格式化
date.format('YYYY年MM月DD日 HH:mm:ss.SSS 星期e a');
// => "2014 年"
date.format('YYYY 年');
// => "2014 年"

// 自定义格式化
var month = {
   "01": "January",
   "02": "February",
   "03": "March",
   "04": "April",
   "05": "May",
   "06": "June",
   "07": "July",
   "08": "August",
   "09": "September",
   "10": "October",
   "11": "November",
   "12": "December",
};
date.format('YYYY年MM月DD日 HH:mm:ss.SSS 星期e a', {
   // 要替换的字段，以及要替换的值
   'MM': month
});
// => "2014年October月04日 17:28:06.363 星期六 下午"

Parameters/Arguments:
Name Type Argument Description

  Object
Example
// this

// in

date.range('in 1 week');
// 今天 - 下个月今天
date.range('in 1 month');
// 今天 - 明年今天
date.range('in 1 year');
