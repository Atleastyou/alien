<inner> emit(context, eventType, arg)

事件触发，只要有一个事件返回false，那么就返回false，非链式调用
Parameters/Arguments:
Name Type Argument Description
context   Object 指定上下文，默认为 Emitter 实例对象
eventType   String |   Object <optional>
 事件类型，多个事件类型使用空格分开
arg   * <repeatable>
 事件传参，多个参数依次即可
Returns:
  * 函数执行结果
Example
var emitter = new Emitter();
emitter.emit('hi', 1, 2, 3);
emitter.emit('hi', 1, 2);
emitter.emit('hi', 1);
emitter.emit('hi');
emitter.emit(window, 'hi');
emitter.emit(window, 'hi', 1);
emitter.emit(window, 'hi', 1, 2);
emitter.emit(window, 'hi', 1, 2, 3);
