<static> format(format, date, config)

格式化日期
主要参考ECMA规范定义：YYYY-MM-DDTHH:mm:ss.sssZ
其他参数参考自moment
Parameters/Arguments:
Name Type Argument Description
format   String 格式化字符串
假设当前时间为：2014年1月1日 19点9分9秒9毫秒 周三
日期
至少4位年份YYYY 2014
至少2位年份YY 14
至少2位月份MM 01
至少1位月份M 1
至少2位日期DD 01
至少1位日期D 1
时间
至少2位24小时制小时HH 19
至少1位24小时制小时H 19
至少2位12小时制小时hh 07
至少1位12小时制小时h 7
至少2位分钟数mm 09
至少1位分钟数m 9
至少2位秒数ss 09
至少1位秒数s 9
至少3位毫秒数SSS 009
至少2位毫秒数SS 09
至少1位毫秒数S 9
时段
星期e 三
上下午a 下午
date   Date |   Object |   Number |   String <optional>
 日期
config   Object <optional>
 格式配置
Returns:
  null |   string
Example
// 默认的格式化
date.format('YYYY年MM月DD日 HH:mm:ss.SSS 星期e a');
// => "2014 年"
date.format('YYYY 年');
// => "2014 年"

// 自定义格式化
var month = {
   "01": "January",
   "02": "February",
   "03": "March",
   "04": "April",
   "05": "May",
   "06": "June",
   "07": "July",
   "08": "August",
   "09": "September",
   "10": "October",
   "11": "November",
   "12": "December",
};
date.format('YYYY年MM月DD日 HH:mm:ss.SSS 星期e a', {
   // 要替换的字段，以及要替换的值
   'MM': month
});
// => "2014年October月04日 17:28:06.363 星期六 下午"

Parameters/Arguments:
Name Type Argument Description

 被比较时间，默认为当前时间
Returns:

Example
// 过去时间
date.from(Date.now() - 1);
// => "刚刚"
date.from(Date.now() - 10*1000);
// => "10秒前"
date.from(Date.now() - 61*1000);
// => "1分钟前"
date.from(Date.now() - 60*60*1000);
// => "1小时前"
date.from(Date.now() - 24*60*60*1000);
// => "1天前"
date.from(Date.now() - 30*24*60*60*1000);
// => "1个月前"
date.from(Date.now() - 12*30*24*60*60*1000);
// => "1年前"
date.from(Date.now() - 20*12*30*24*60*60*1000);
// => "19年前"
date.from(Date.now() - 100*12*30*24*60*60*1000);
// => "很久之前"

// 将来时间
date.from(Date.now() + 1);
// => "即将"

// => "1小时后"

// => "1天后"
date.from(Date.now() + 30*24*60*60*1000);
// => "1个月后"
date.from(Date.now() + 12*30*24*60*60*1000);
// => "11个月后"
date.from(Date.now() + 20*12*30*24*60*60*1000);
// => "19年后"
date.from(Date.now() + 100*12*30*24*60*60*1000);
// => "98年后"

<static> range(type, ref)

计算日期范围
Parameters/Arguments:
Name Type Argument Description
type   String 可选 `this N days`/`this N weeks`/`this N months`/`this N years` 当日/周/月/年 `in Ndays`/`in N weeks`/`in N months`/`in N years` 几日/周/月/年内
ref   Date <optional>
 起点时间，默认为当前时间
Returns:
  Object
Example
// this
date.range('this 1 day');

// 本月一 - 本月末
date.range('this 1 month');
// 本年头 - 本年尾
date.range('this 1 year');

// in
date.range('in 1 day');
// 今天 - 一周内
date.range('in 1 week');
// 今天 - 下个月今天
date.range('in 1 month');
// 今天 - 明年今天
date.range('in 1 year');
