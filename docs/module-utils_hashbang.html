utils/hashbang

必须以#!开头，后续的 querystring 必须符合标准

<static> un(part, key, listener)

移除监听 hashbang
Parameters/Arguments:
Name Type Argument Description
part   String 监听部分，可以为`query`或`path`
key   String |   Number |   Array |   function <optional>
 监听的键，`query`为字符串，`path`为数值，多个键使用数组表示
listener   function <optional>
 监听回调，回调为空表示删除该键的所有监听队列
Example
// path
// 移除 path 0字段上的一个监听
hashbang.un('path', 0, fn);
// 移除 path 0字段上的所有监听
hashbang.un('path', 0);
// 移除 path 所有字段的一个监听
hashbang.un('path', fn);
// 移除 path 所有字段的所有监听
hashbang.un('path');

// query
// 移除 query abc 键上的一个监听
hashbang.un('query', 'abc', fn);
// 移除 query abc 键上的所有监听
hashbang.un('query', 'abc');
// 移除 query 所有键上的一个监听
hashbang.un('query', fn);
// 移除 query 所有键上的所有监听
hashbang.un('query');
