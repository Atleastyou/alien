Name Type Argument Description

 x轴终点

 y轴终点
options   Object <optional>
 配置
Properties
Name Type Argument Description
duration   Object <optional>
 动画时间，默认789，单位 ms
easing   Object <optional>
 动画缓冲，默认 swing 默认配置的缓冲函数有：linear、easeIn、easeOut、easeBoth、easeInStrong、easeOutStrong easeBothStrong、easeOutQuart、easeInOutExpo、easeOutExpo、swing、swingFrom、swingTo backIn、backOut、bounce、doubleSqrt，也可以自定义缓冲函数，只有一个参数t，表示时间比 = 已耗时 / 总时间， 计算值=开始值 + ( 结束值 - 开始值 ) * 时间比
delay   Object <optional>
 动画延时，默认0，单位ms
callback   function <optional>
 回调

  undefined
Example

   y: 100
}, {
   duration: 1000,

   delay: 100

   alert('OK');
