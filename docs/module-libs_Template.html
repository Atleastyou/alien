Module: libs/Template

libs/Template

new require(libs/Template)(options)

模板引擎
注意点：不能在模板表达式里出现开始或结束符，否则会解析错误
1. 编码输出变量
{{data.name}}
2. 取消编码输出变量
{{=data.name}}
3. 判断语句（if）
{{if data.name1}}
{{else if data.name2}}
{{else}}
{{/if}}
4. 循环语句（list）
{{list list as key,val}}
{{/list}}
{{list list as val}}
{{/list}}
5. 过滤（|）
第1个参数实际为过滤函数的第2个函数，这个需要过滤函数扩展的时候明白，详细参考下文的addFilter
{{data.name|filter1|filter2:"def"|filter3:"def","ghi"}}
6. 反斜杠转义，原样输出
\{{}} => {{}}
Parameters/Arguments:
Name Type Argument Description
options   Object <optional>
 配置
Properties
Name Type Argument Default Description
compress   Boolean <optional>
 true 是否压缩，默认为 true
Example
var tpl = new Template('{{name}}');
tpl.render({name: 'yundanran'});
// => 'yundanran'
